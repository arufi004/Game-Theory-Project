# Game-Theory-Project
This project was created for the CAP5507 - Game Theory Course during the Spring 2026 term. 
This project allows the user to either input payoffs for a game, or allow the program to generate the payoffs for a game. 
It then takes this data and prints the normal form, nash equilibria, strategies and payoffs, best responses, and indifference mix probabilities.
I certify that this work is my own and that none of it is the work of any other person.
The professor's instructions required the assignment to be called GT-SPR2020, hence the folder name.

The project only contains a single class: The Controller class, which contains all the methods used in this program.
